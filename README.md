# Onboarding Tracker Web App

Система управления процессом адаптации новых сотрудников. Позволяет администраторам создавать планы обучения, менторам — отслеживать прогресс, а сотрудникам — отмечать выполнение этапов.

## Стек технологий
* **Backend:** Go (Golang) `
* **Database:** PostgreSQL 
* **Frontend:**  JavaScript, HTML, CSS
* **DevOps:** Docker

## Функционал
- **Управление планами:** Создание индивидуальных программ обучения на основе библиотеки материалов.
- **Автоматизация статусов:** Система автоматически переводит план в статус "В процессе" при начале работы и в "Завершен", когда все задачи выполнены.
- **Ролевая модель:** - `Admin`: Полный доступ, создание планов.
    - `Mentor`: Просмотр и комментирование планов своих подопечных.
    - `Employee`: Отметка о выполнении своих задач.
- **Фильтрация:** Умные фильтры по сотрудникам и текущему состоянию планов.
